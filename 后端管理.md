## Chunk 1 — 🚨 P0 | 数据持久化 (localStorage)

> 预估工时：1h | 文件：admin-system/index.html
> 现状：内存数组，刷新丢数据，新增/编辑/删除全部白做。

### 任务

给 `products`、`orders`、`stories`、`banners` 四个数据源加上 localStorage 读写层。

### 具体方案

#### 1.1 封装数据读写工具函数（在现有 JS 中添加）

```javascript
// ==================== localStorage 持久化层 ====================
const STORAGE_KEYS = {
  products: 'rotorua_admin_products',
  orders: 'rotorua_admin_orders',
  stories: 'rotorua_admin_stories',
  banners: 'rotorua_admin_banners',
}

function loadData(key, fallback) {
  try {
    const saved = localStorage.getItem(STORAGE_KEYS[key])
    if (saved) {
      const parsed = JSON.parse(saved)
      if (Array.isArray(parsed) && parsed.length > 0) return parsed
    }
  } catch (e) { /* 忽略解析错误，返回默认值 */ }
  return fallback
}

function saveData(key, data) {
  localStorage.setItem(STORAGE_KEYS[key], JSON.stringify(data))
}
```

#### 1.2 修改数据初始化

将原来的：

```javascript
const products = [...]
const orders = [...]
const stories = [...]
const banners = [...]
```

改为：

```javascript
// ==================== 数据（优先从本地存储加载） ====================
const DEFAULT_PRODUCTS = [...]
const DEFAULT_ORDERS = [...]
const DEFAULT_STORIES = [...]
const DEFAULT_BANNERS = [...]

let products = loadData('products', DEFAULT_PRODUCTS)
let orders = loadData('orders', DEFAULT_ORDERS)
let stories = loadData('stories', DEFAULT_STORIES)
let banners = loadData('banners', DEFAULT_BANNERS)
```

#### 1.3 在所有 CRUD 操作后同步写入

- `saveProduct()` → 调用 `saveData('products', products)`
- `deleteProduct()` → 调用 `saveData('products', products)`
- `saveStory()` → 调用 `saveData('stories', stories)`
- 后续订单状态变更同理

#### 1.4 增加「重置数据」功能（调试用）

在设置页底部或侧边栏底部加一个「重置数据到初始状态」按钮，带确认弹窗：

```javascript
function resetAllData() {
  if (!confirm('确认重置所有数据？已修改的数据将丢失。')) return
  ;['products','orders','stories','banners'].forEach(key => {
    localStorage.removeItem(STORAGE_KEYS[key])
  })
  products = [...DEFAULT_PRODUCTS]
  orders = [...DEFAULT_ORDERS]
  stories = [...DEFAULT_STORIES]
  banners = [...DEFAULT_BANNERS]
  renderAll()
  showToast('🔄 已重置为初始数据')
}
```

- [x] 新增一个产品 → 刷新页面 → 产品还在
- [x] 编辑一个产品 → 刷新页面 → 修改生效
- [x] 删除一个产品 → 刷新页面 → 确实删除
- [x] 新增/编辑/删除故事后刷新数据不丢

```javascript
    const dateStr = `${d.getMonth()+1}/${d.getDate().toString().padStart(2,'0')}`
    last7Days.push(dateStr)
    const dayStr = d.toISOString().slice(0,10) // "2026-06-28"
    // 注意：订单日期格式是 "2026-06-28"，需要统一格式
    counts.push(orders.filter(o => o.date === dayStr).length)
  }
  new Chart(ctx, {
    type: 'bar',
    data: {
      labels: last7Days,
      datasets: [{
        label: '订单数',
        data: counts,
        backgroundColor: '#0B4F4A',
        borderRadius: 4,
      }]
    },
    options: {
      responsive: true,
      plugins: { legend: { display: false } },
      scales: { y: { beginAtZero: true, ticks: { stepSize: 1 } } }
    }
  })
}
```

```javascript
function showToast(message, type = 'success') {
  const container = document.getElementById('toastContainer')
  const toast = document.createElement('div')
  toast.className = 'toast'
  toast.textContent = message
  toast.style.cssText = `
    padding:10px 18px;border-radius:8px;background:${type === 'success' ? '#16A34A' : '#DC2626'};
    color:#fff;font-size:13px;font-weight:500;box-shadow:0 4px 12px rgba(0,0,0,.15);
    opacity:0;transform:translateX(40px);transition:all .3s;
  `
  container.appendChild(toast)
  requestAnimationFrame(() => { toast.style.opacity = '1'; toast.style.transform = 'translateX(0)' })
  setTimeout(() => {
    toast.style.opacity = '0'; toast.style.transform = 'translateX(40px)'
    setTimeout(() => toast.remove(), 300)
  }, 2000)
}
```
